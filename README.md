# Dars Pro 📚

Arab tili o'qituvchisi uchun professional dars rejalashtiruvchi ilova.

## Funksiyalar
- ✅ Haftalik takrorlanadigan dars jadvali
- ✅ Galma-gal jadval (toq/juft hafta)
- ✅ Bildirishnomalar (ilova yopiq bo'lsa ham)
- ✅ 5 daqiqa oldin eslatma
- ✅ 45 daqiqa keyin "O'tdimi?" so'rovi
- ✅ Statistika va grafiklar
- ✅ Oylik maosh hisoblash (dars uchun va oylik)
- ✅ Ma'lumotlar hech qachon o'chirmaydi
- ✅ Ovoz tanlash (6 xil)
- ✅ Telegram bot integratsiyasi (keyingi versiya)

## Build qilish (Codemagic orqali)

1. GitHub ga yuklang
2. codemagic.io ga kiring (GitHub bilan)
3. "Add app" → bu repositoryni tanlang
4. Flutter → Android → Build
5. APK yuklab oling

## Flutter versiyasi
- Flutter 3.19+
- Dart 3.0+
